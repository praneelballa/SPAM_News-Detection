# SPAM_News-Detection
A machine learning-based web app that detects spam or fake text using NLP and real-time predictions.

## Features
- Text classification (Spam / Not Spam)
- Confidence score
- Simple UI using Streamlit

## Tech Stack
- Python
- Scikit-learn
- NLTK
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
